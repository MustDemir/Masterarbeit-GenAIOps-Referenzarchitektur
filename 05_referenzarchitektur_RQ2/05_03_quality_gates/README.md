# 5.3 Quality Gates

Dieser Ordner bildet den fachlichen Kern des Quality-Gate-Kontrollsystems ab.

- `gate_template.yaml`: zentrales Template fuer Gate-Spezifikationen
- `compliance/`: regulatorische Gates
- `technical/`: technische Gates
- `strategic/`: strategische Gates
- `images/`: section-spezifische Visuals

Rego-Entwuerfe, Literaturvergleiche und Methodenartefakte gehoeren nur dann hierher, wenn sie direkt als 5.3-Artefakt verwendet werden. Sonst bleiben sie in `../arbeitsmaterial/`.
