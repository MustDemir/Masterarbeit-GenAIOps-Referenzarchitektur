# 5.2 Architekturuebersicht

Dieser Ordner enthaelt die abschnittsbezogene Struktur fuer die Architekturvisualisierung und den Architekturueberblick.

- `images/final/`: finale, zitierfaehige Abbildungen
- `images/work/`: Arbeitsstaende
- `images/legacy/`: abgeloeste oder historische Varianten

Freie Notizen, Recherche und Kapitel-Drafts bleiben in `../arbeitsmaterial/`.
