# 2.7 Synthese Forschungsluecke

Dieser Ordner ist fuer abschnittsbezogene Artefakte zur Synthese und Forschungsluecke gedacht.

- Abschnittsnahe Synthesetabellen oder konsolidierende Notizen duerfen hier liegen
- Kapitelweite Analysen und Vorarbeiten bleiben in `../arbeitsmaterial/`
- Ziel ist eine knappe, kapitelabschliessende Verdichtung der Vorabschnitte
